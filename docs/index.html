Cotton grows more root — and mounts a calmer stress response — in spaceflight

Targeting Improved Cotton Through Orbital Cultivation: root architecture and the root/shoot transcriptome of wild-type and AVP1-overexpressing (AVP-OX) cotton aboard the ISS, analysed through a fully reproducible pipeline.

GitHub repository Manuscript draft Project roadmap

The finding

Against a shared, strongly tissue-specific spaceflight response (a canonical hypoxia/wounding induction with translational suppression in wild-type roots), the AVP-OX engineering acts as an attenuator, not an amplifier of the defence/stress programme — while the engineered lines sustain a larger primary-root spaceflight response than wild type. In short: AVP-OX cotton appears to experience microgravity as less stressful while growing more root — a favourable trait for space agriculture.

Supported by four independent analyses:

Morphometrics
AVP-OX larger primary-root flight response (P ≤ 0.04)
GO / KEGG
Jasmonate/wounding defence suppressed in AVP-OX
PhysioSpace
Lower osmotic/wounding stress scores vs WT
WGCNA
Defence module down in AVP-OX; growth-coupled module up

Reproducible pipeline

Every result regenerates from committed scripts (R 4.6; versions in ENVIRONMENT.txt).

  1. deseq2/ — DESeq2 contrasts + factorial (V4) model
  2. crosswalk/ — Gohir → Arabidopsis ortholog map (CottonGen BLASTP)
  3. go_analysis/ — clusterProfiler GO / KEGG
  4. physiospace/ — PhysioSpace stress-pattern decoding
  5. wgcna/ — co-expression modules (GO-named)
  6. morphometrics/ — RSML trait extraction + mixed models
  7. integration/ — transcriptome ↔ root-architecture model

Honest limitations

  • The large total-length flight effect is lateral-root driven; absolute magnitude is pending Flight/Ground image-scale calibration. The primary-root AVP-OX result is robust to this.
  • RNA-seq libraries are not individually paired to imaged plants, so the expression ↔ architecture integration is group-level (n = 6) and correlational — pathways are consistent with driving growth, not proven causal.
  • Ortholog mapping is best-BLASTP-hit (approximate); single mission.
